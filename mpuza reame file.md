# <a name="mpuza-login-form"></a>Mpuza Login Form
A simple, clean **HTML & CSS login form** design for a web application called **Mpuza**. This project demonstrates basic front‑end skills including layout, form styling, icons, and page structure.

-----
<a name="preview"></a>The login form includes: - A top navigation bar - Centered login card with logo - Email and password inputs with icons - “Remember me” checkbox - Sign‑in button - Fixed footer

-----
## <a name="technologies-used"></a>Technologies Used
- **HTML5**
- **CSS3**
- **Font Awesome (CDN)** for icons
-----
## <a name="project-structure"></a>Project Structure
mpuza-login-form/\
│\

├── login.css              # Styling file\

-----
## <a name="how-to-use"></a>How to Use
1. Download or clone this repository
1. Make sure all files are in the same folder
1. Open **mpuza login form.html** in any modern web browser
-----
## <a name="features"></a>Features
- Responsive input layout
- Icons inside input fields
- Clean and simple UI
- Fixed footer at the bottom
- Easy to customize
